Projeto Integrador Fatec

O Projeto Integrador é um projeto propostosto pela Fatec Zona Leste para os alunos do curso de desenvolvimento
de software multiplataforma. Em tal projeto os alunos foram encorajados a entrar em contato com pequenas
empresas e projetos na região da Zona Leste de São Paulo para desenvolver um website para tal instituição.
